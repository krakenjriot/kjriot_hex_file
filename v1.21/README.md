# v1.21 Hex File (release date 4/10/2020)
- Fixed Sensor Inaccurate readings
- Removed pin number 4,5,17 and 18

DHT11 = v1.21.RUH.DHT11P5
DHT21 = v1.21.RUH.DHT21P5
DHT22 = v1.21.RUH.DHT22P5
